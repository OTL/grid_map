sandbox for implementation of grid_map and navigation related code.

## Pre-requirements

On Ubuntu:

```bash
sudo apt-get install g++ pkg-config libx11-dev libasound2-dev libudev-dev protobuf-compiler
```

## Try examples

```bash
cargo run --release --example dwa_gui
```
